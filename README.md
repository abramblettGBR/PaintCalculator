# GBR Paint Calculator

A simple web-based tool for estimating paint needs, including primer, walls, ceilings, trim, and doors.

## How to Use
- Open `index.html` in any browser.
- Add rooms, enter dimensions, and calculate.

## Live Demo
[View the app here](https://yourusername.github.io/gbr-paint-calculator/) (replace with your actual URL after Step 5).

## Features
- Multi-room support
- Primer toggle
- Door painting option (matches trim)
